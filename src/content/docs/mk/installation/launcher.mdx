The next thing on the list is a launcher designed for playing Minecraft with mods, you can use the official launcher, however,
it is not optimized well for a modded environment and lacks a lot of core features and functionalities unlike other launchers.

## Choosing a launcher

Some of the most popular launchers for modded Minecarft are:

- [The Prism Launcher](https://prismlauncher.org/)
- [The CurseForge Launcher](https://curseforge.com/)
- [The ATLauncher](https://atlauncher.com/)
- [The Modrinth Launcher](https://modrinth.com/app)
- [MultiMC](https://multimc.org/) (OUTDATED)

*Any of the first 3 launchers usually provide the best experience when it comes to playing modded Minecraft.*

---
title: Prequirements
description: Requirements in order to be able to join the server.
---

Before you can install the modpack and join the server, be sure that
you have everything in the list below so that you can continue.

- A non-cracked (legitimate) copy of Minecraft: Java Edition. (REQUIRED)
- A Discord Account in order to link your Minecraft Account. (REQUIRED)
- At least 4GB of available storage on your Main Drive. (REQUIRED)
- An up-to-date OS, such as Windows 10 or Windows 11. (STRONGLY RECOMMENDED)
- A minimum of 8GB of RAM for a playable experince. (STRONGLY RECOMMENDED)

Once you have checked that you have all of the prerequirements on the list, you can continue with the Installation Guide.
